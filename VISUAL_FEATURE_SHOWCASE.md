## 🔄 Animation Flows

### Reaction Animation Flow
```
1. User clicks reaction button
   ↓
2. Reaction menu appears (scale + fade)
   ↓
3. User selects reaction type
   ↓
4. 15 particles burst outward
   ↓
5. Particles fade while moving
   ↓
6. Count updates with pop animation
   ↓
7. Reaction badge appears
```

### Milestone Detection Flow
```
1. User writes post with keyword
   ↓
2. System detects "promoted"
   ↓
3. Trophy badge added automatically
   ↓
4. Confetti animation triggers
   ↓
5. Gradient border applied
   ↓
6. Pulsing effect active
```

### Smart Suggestion Flow
```
1. User types 50+ characters
   ↓
2. 500ms debounce delay
   ↓
3. AI analyzes content
   ↓
4. Keywords extracted
   ↓
5. Hashtags suggested
   ↓
6. User clicks to add
   ↓
7. Hashtag inserted at cursor
```
