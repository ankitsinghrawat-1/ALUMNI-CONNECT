Technical Implementation:

1. Default State (Icon Only)

  • Button width: 40px (fixed)
  • Only icon visible
  • Text has opacity: 0 and translateX(-10px)

2. Hover State (Expanded)

  • Button width: auto (expands to fit content)
  • Padding adjusts for text
  • Text opacity: 1 and translateX(0)
  • Transition: 0.3s cubic-bezier(0.4, 0, 0.2, 1)

3. Sibling Behavior

  • Flexbox with gap: 0.75rem
  • Siblings automatically reflow to accommodate expansion
  • Smooth transitions on all buttons
  • No JavaScript needed - pure CSS!
